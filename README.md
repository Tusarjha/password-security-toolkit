# Password Security Toolkit

## Overview

Password Security Toolkit is a command-line Python application designed to analyze and improve password security. The toolkit provides multiple utilities, including password strength analysis, common password detection, entropy estimation, password policy validation, and secure password generation.

This project was developed to understand basic cybersecurity concepts related to password security and authentication.

---

## Features

### 1. Password Strength Analysis

Evaluates password strength based on:

* Password length
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters

### 2. Common Password Detection

Checks whether a password belongs to a list of commonly used and easily guessable passwords.

### 3. Entropy Estimation

Provides a simple entropy score based on password length and character diversity.

### 4. Password Policy Validation

Validates whether a password satisfies common security requirements:

* Minimum length
* At least one uppercase letter
* At least one numeric digit

### 5. Secure Password Generator

Generates random passwords containing:

* Uppercase letters
* Lowercase letters
* Numbers
* Special characters
